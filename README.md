# Velha

> A simple tic-tac-toe game built with React.js.

Play it [HERE](https://theuves.github.io/velha).

## Development

Do the following:

```
$ git clone https://github.com/theuves/velha
$ cd velha
$ npm install
$ npm start
```

After this, you can access it at *http://localhost:5000*.
